The Legislative Tracker

The Legislative Tracker web app is designed to assist in tracking which members of a given legislature have signed on to sponsor a given bill supported by the Communications Workers of America.

The app's code is freely available under the Free Software Foundation's GNU General Public License 3.0 and can be found on GitHub here.
